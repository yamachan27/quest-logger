# Quest Logger 🎮

RPG風のタスク管理アプリ。クエストをクリアしてレベルアップしよう！

## 特徴
- 🎯 RPG風のレベルアップシステム
- 🏆 15種類の称号コレクション
- 🔥 連続達成でストリークボーナス
- 📅 カレンダーで過去の達成履歴確認
- 🎁 毎日のログインボーナス
- ✨ クエスト完了時のアニメーション

## 使い方
1. クエストを追加
2. 難易度を選択（簡単/普通/難しい）
3. 完了したらチェック
4. XPを獲得してレベルアップ！

## 技術スタック
- React
- Tailwind CSS
- LocalStorage
- Lucide React Icons
